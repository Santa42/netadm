Основы сетей
############

Данный конспект собран в помощь прохождения курса Сетевой инженер.
Этот проект помогает понять и разобрать сетевые технологии, которые используются в большинстве современных сетей.
В документацию добавлены задачи с курса для отработки различных тем.
На протяжении всей документации есть основная ветка заданий, которые в итоге формируют итоговый практический проект, включающий в себя все пройденные темы.
И есть дополнительные задания со \* для тех, кому хочется отработь работу с различными протоколоми более глубоко
